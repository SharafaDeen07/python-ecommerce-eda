Project Title
# Ecommerce Sales Analysis with Python

Project Overview
This project analyzes ecommerce sales data for a Jumia/Konga-style online retail business.

The objective is to identify sales trends, customer behavior, product performance, and business opportunities using Python and data visualization.

Business Questions
- What product categories sell best?
- What is the revenue trend over time?
- Which customers generate the highest revenue?
- What is the average order value?
- Which products perform poorly?
- What recommendations can improve business performance?

Dataset
The dataset contains ecommerce transactions including:

- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit
- Customer ID (sythenthic data was not included in the dataset)
- Customer Name (sythenthic data was not included in the dataset)

Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

Key Insights
### Top Performing Categories

Electronics generated the highest sales revenue.

### Revenue Trend

Revenue increased steadily throughout the year with seasonal spikes during promotional periods.

### High-Value Customers

A small segment of customers contributed a significant share of total revenue.

### Low-Performing Products

Several products consistently generated low sales and may require strategic review.

Business Recommendations
## Business Recommendations

1. Increase investment in top-performing categories.
2. Introduce loyalty programs for high-value customers.
3. Bundle low-performing products with popular items.
4. Improve inventory planning using historical sales trends.
5. Increase average order value through cross-selling and upselling.
